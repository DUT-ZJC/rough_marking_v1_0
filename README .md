# rough_marking v1.0（MVP / 可运行 Demo）

本项目用于“毛坯自动定位 + 基准约束配准”的最小可运行版本，实现以下闭环：

- 输入：CAD 理想模型（STEP）+ 实际扫描毛坯（STL 三角网格）
- 粗配准：点云全自动粗配准（不依赖人工基准）
- 特征识别：
  - CAD：从 STEP 解析提取平面/圆柱等解析特征（按现有实现）
  - Scan：从 STL 网格解析/分割拟合平面/圆柱特征（按现有实现）
- 交互选择：双窗口界面
  - 左窗口：CAD（灰色底模 + 彩色特征）
  - 右窗口：Scan（灰色底模 + 彩色特征）
  - 鼠标点击选中特征（选中高亮红色），并在右侧面板中添加/删除约束
- 精配准：约束嵌入点到面 ICP 的优化求解（平面/圆柱约束均可参与）
- 输出：最终刚体变换矩阵 T（CAD->Scan）+ 配准结果可视化

---

## 1. 快速开始

### 1.1 环境准备
建议 Python 3.10+。

### 1.2 安装依赖
在项目根目录执行：

```bash
pip install -r requirements.txt
```

> 注意：读取 STEP 依赖 pythonocc-core（Windows / Linux 一般可装）。
> 若安装困难，可临时将 STEP 转 STL 后加载（功能会受影响，建议尽量装 pythonocc-core）。

### 1.3 配置输入路径
编辑 `config.py`，修改：
- `CAD_STEP_PATH`：STEP 文件路径
- `SCAN_STL_PATH`：STL 文件路径

### 1.4 运行
```bash
python main.py
```

运行后流程如下：
1) 弹出粗配准结果窗口（红：CAD 点云；灰：Scan 点云），关闭后继续
2) 弹出双窗口选择界面：
   - 左侧 CAD 选特征，右侧 Scan 选特征
   - 在右侧面板选择约束类型（plane-plane / cyl-cyl）、容差，并 Add Constraint
   - 可重复添加多条约束；支持 Remove Last 删除最后一条；Done 后进入优化
3) 优化结束后弹出最终配准结果窗口

## 2. 交互说明（双窗口约束选择）

### 2.1 颜色说明（建议）
- 灰色：底模（未识别为特征的区域）
- CAD 特征：平面（绿色）、圆柱（蓝色）
- Scan 特征：平面（黄色）、圆柱（紫色）
- 选中高亮：红色（覆盖显示）

### 2.2 操作步骤（推荐）
1) 在 CAD 左窗口点击一个基准特征（面/孔）
2) 在 Scan 右窗口点击对应的扫描特征
3) 在右侧面板选择约束类型与容差（例如 0.1° / 0.1mm）
4) 点击 Add Constraint，将该 CAD↔Scan 配对加入约束列表
5) 重复添加多条约束（建议至少 2 个互补约束，如“一个面 + 一个孔”）
6) 点击 Done 开始约束优化

## 3. 输出

- 终端输出：
  - 选定约束列表
  - 最终变换矩阵 `T_final`（CAD->Scan）
  - 迭代收敛信息（rmse、步长等）
- 图形输出：
  - 粗配准可视化窗口
  - 精配准最终结果可视化窗口

## 4. 常见问题（简要）

1) STEP 无法读取：
- 请确认已安装 `pythonocc-core`
- 或临时将 STEP 转 STL

2) 特征数量过多导致界面卡：
- 可在 Scan 特征提取后做 TopN 筛选（按面积、拟合残差等）

3) 优化不收敛：
- 尝试放宽容差（如 0.3° / 0.3mm），先稳定后再逐步收紧
- 检查粗配准是否明显翻转/跑飞（粗配准窗口可观察）
